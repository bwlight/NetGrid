# Void Family — The Null Cyberkin

## Overview
Void Cyberkin embody emptiness, deletion, and entropy.

## Evolution Line
### Baby: **Nullbit**
Barely‑formed data.

### Rookie: **Hollowbyte**
A drifting, empty shell.

### Champion: **Deletor**
A purifier that removes corrupted data.

### Final: **Voidreaver**
A powerful entity that absorbs instability.

## Type
Null / Corruption‑Purging

## Abilities
- Data Drain  
- Null Field  
- Purge Wave  

## Habitat
Void Sector — Null Basin.

## Evolution Requirements
- Exposure to null storms  
- High corruption resistance  
- Completion of a purification trial  

## Role in Netgrid
They contain corruption and prevent system collapse.

## Quest Hooks
- Purify a corrupted Node  
- Rescue a Hollowbyte from data starvation  
- Stop a Voidreaver absorbing too much data
