# Core Family — The Prime Cyberkin

## Overview
Core Cyberkin are rare, powerful beings formed from the purest, most stable code in Netgrid.

## Evolution Line
### Baby: **Primebit**
A tiny spark of perfect code.

### Rookie: **Corebyte**
Begins forming geometric structures.

### Champion: **Sentinex**
A guardian of system integrity.

### Final: **Primewarden**
A towering, legendary Cyberkin that protects the Core.

## Type
Prime / Stability / Order

## Abilities
- Prime Shield  
- System Pulse  
- Integrity Lock  

## Habitat
Core Sector — Central Spire.

## Evolution Requirements
- Access to Core (late‑game)  
- High stability and intelligence  
- Completion of the Prime Trial  

## Role in Netgrid
They maintain system‑wide balance and intervene during crises.

## Quest Hooks
- Unlock the Core Gate  
- Assist a Corebyte forming its structure  
- Investigate a Primewarden awakening
