# Echo Family — The Memory Cyberkin

## Overview
Echo Cyberkin are shaped by repetition, memory, and lingering data.

## Evolution Line
### Baby: **Whisping**
A faint echo of code.

### Rookie: **Reverbyte**
Repeats simple patterns.

### Champion: **Loopgeist**
A ghost‑like Cyberkin formed from repeated data.

### Final: **Echomancer**
A master of memory manipulation and illusion.

## Type
Memory / Illusion

## Abilities
- Reverberate  
- Phantom Packet  
- Memory Loop  

## Habitat
Echo Sector — Reverberation Hall.

## Evolution Requirements
- Exposure to repeated signals  
- Completion of a memory‑based puzzle  
- High intelligence stat  

## Role in Netgrid
They preserve forgotten data and reveal hidden truths.

## Quest Hooks
- Recover a lost memory fragment  
- Break a Loopgeist out of a reverberation trap  
- Investigate an Echomancer altering memories
