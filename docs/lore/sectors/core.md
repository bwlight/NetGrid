# Core Sector

## Overview
The central nexus of Netgrid. All Sectors converge here, but few may enter.

## Origin
Formed from the densest concentration of stable code during Netgrid’s birth.

## Function
Acts as the system’s command center, regulating balance and stability.

## Atmosphere
Bright, geometric, impossibly ordered. Feels like standing inside pure logic.

## Hazards
- Integrity locks
- System guardians
- High‑level corruption barriers

## Gate Node
`core-gate` (currently sealed)

## Stream Node
`core-stream` (currently sealed)

## Neighbor Sectors
None — Core is isolated.

## Cyberkin Affinity
Prime‑class Cyberkin and system guardians.

## Notable Locations
- The Prime Lattice
- The Central Spire
- The System Heart

## Quest Hooks
- Unlock the Core Gate
- Investigate a system‑wide stability fluctuation
- Encounter a Prime‑class Cyberkin
